### 🚀 Highlighted Project

#### 👤 **Student Face Recognition Attendance Management System**
A complete desktop application built for school and teacher attendance tracking.

* **Core Features:**
  * **Role-Based Security:** Admin and Teacher role access control.
  * **Real-Time Recognition:** InsightFace & OpenCV models tracking webcam video input.
  * **Biometric Registration:** Register multiple photos per user with verification checks.
  * **Analytics Dashboard:** Graphical attendance reports, session lists, and export modules (CSV, Excel, PDF).
* **Tech Stack:** Python, CustomTkinter, SQLite, InsightFace, OpenCV, PyInstaller.
